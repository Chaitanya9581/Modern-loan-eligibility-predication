# 🏦 Modern Loan Eligibility Prediction

A Machine Learning-based web application that predicts whether a loan applicant is eligible for a loan based on financial and personal information. This project helps automate the loan approval process by providing quick and accurate predictions.

---

## 📌 Project Overview

The main objective of this project is to predict loan eligibility using Machine Learning algorithms. The application analyzes applicant details such as income, education, employment status, loan amount, credit history, and other factors to determine whether the loan is likely to be approved.

---

## 🚀 Features

- Loan eligibility prediction using Machine Learning
- User-friendly interface
- Data preprocessing and feature encoding
- Model training and evaluation
- Fast and accurate predictions
- Easy to use and deploy

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Chaitanya9581/Modern-loan-eligibility-predication.git
```

Navigate to the project directory:

```bash
cd Modern-loan-eligibility-predication
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

or (if using Streamlit)

```bash
streamlit run app.py
```
